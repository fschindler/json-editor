# Stories for [@sagold/react-json-editor](https://github.com/sagold/json-editor/tree/main/packages/react-json-editor)

> [@sagold/react-json-editor](https://github.com/sagold/json-editor/tree/main/packages/react-json-editor) is an implementation of [headless-json-editor](https://github.com/sagold/json-editor/tree/main/packages/headless-json-editor) using [semantic-ui](https://semantic-ui.com/) for components.
> `react-json-editor` is currently in a very early state, but the package is published and regularly tested

installation

`yarn add @sagold/react-json-editor`
